What is ownCloud?

ownCloud is an Internet based synchronization tool that allows you to automatically sync up your Accion files saved locally on your computer hard drive to an Accion file server.


How does ownCloud Work?

ownCloud is a server/client software. When an Accion employee is setup with ownCloud, the client software is installed o his/her computer. That client software is set to automatically startup when the computer is turned on, and sync new or modified files deposited in the ownCloud directory every time it detects changes, and if internet connection is present. The ownCloud client software can be installed in multiple computers so you can access your synched files from any computer where the ownCloud client is installed.


What are the benefits of using ownCloud?

• Automatic sync of your local files to the ownCloud server
• Transparently updates your files to multiple computers. ( laptop, and Desktop )
• Keep an online copy of your local Accion files. In the event of a computer hard failure or computer theft you can restore your files from your online copy.
• Access your files from anywhere. Once your files are synched to the ownCloud server, you can access those files from anywhere either through the ownCloud client, or through a web browser.


Can I Access My files From the Web?

Yes. ownCloud provides a secure way of accessing your files while on the go without using the ownCloud client software.


Are my files secure?

Yes. Before your files are transmitted across the Internet, they are encrypted with a 256-bit encryption. The contents and the names of the files are not known while stored on the ownCloud server.


Can ownCloud replace my home Drive?

Yes. You can save all of your Accion related files in your ownCloud folder. Home Drives will still exist for Boston, DC and Bogota users, for network document scanning purposes.


What file formats are allowed to sync to ownCloud?

You can sync most file formats including work related .JPGs and .PNG image files. Please restrain from syncing big (1 GB or more) files. SQL and Microsoft Access databases are NOT allowed to sync because of their self contained nature.


Can I dump my Email Archive file in ownCloud?

No. ownCloud won’t sync .PST files. We commend you backup your PST files to external devices like External hard drives, or DVDs so you have a copy of your entire mail archive in case your computer crashes or gets stolen.



How much storage space to do I get with ownCloud?

Currently you are allowed up to 20 GB of storage space.

Sharing a File or Folder in ownCloud


Yes, you can share a folder or file in ownCloud through the web interface only .There is no option to share files or folders through the computer client at this moment. To share a file or folder please login to your ownCloud account by typing https://owncloud.accion.org on your preferred browser.

Login using your Accion user-name and password i.e. the same password you use for Outlook.

Once you're logged in you will see all the stuff you have uploaded to your ownCloud account. Hover your mouse over the folder or file that you want to share, and you will get a dynamic menu with the rename, download or Share options:

click on the “Share” button and a small menu will pop-up:

if the person you are sharing the folder or file with is an Accion Employee, type the first name of that person and a list of all the names of employees with that name will pop-up. Choose the right name and select it. By default the person will only be allowed to download whatever whatever you are sharing. If you want that person to be able to edit or upload more files to the folder, please editg the permissions by clicking on the edit button:

If the person you are sharing with is not an Accion employee, you can share using an external link. click on the Share with link and type the email address of the person on the email field:

From that window you can password protect whatever you are sharing, and set a expiration date for the link as well.
